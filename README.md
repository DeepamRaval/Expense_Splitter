Splito is a website that makes splitting bills and keeping track of money effortless. 
You can log expenses, assign who owes what, and settle up with clear balances and reminders. 
Perfect for roommates, trips, dinners or group gifts.

Features
	•	Add multiple participants and total expenses
	•	Automatically splits the amount equally
	•	Stores balances securely using MySQL
	•	Lets you view all balances and clear all data
	•	Simple, clean, and responsive interface
Languages Used:

  •	Frontend: HTML, CSS, JavaScript
	•	Backend: Python (Flask)
	•	Database: MySQL

Project Structure:
expense_splitter/
│
├── app.py                 # Main Flask backend
├── templates/
│   ├── starting.html      # Landing page
│   ├── index.html         # Expense input page
│   ├── balances.html      # Displays all balances
│   ├── result.html        # Success message page
